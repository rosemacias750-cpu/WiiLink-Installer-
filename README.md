an app that simplifies the install of WiiLink and some other things that may be useful (only for linux)
Features:
1: Install WiiLink Assets (or copy them to a path)
2: Search & Download Games (going away soon)
3: Set Custom Path
4: Local ISO to WBFS
5: Exit
how to run this: 
Step 1: run pip install requests beautifulsoup4 psutil pywin32 winshell in the Linux terminal 
Step 2: Give permissions to run as a program 
Step 3: Run and Enjoy!
(P.S. Next version will include as Windows version. This was made by AI which we consider open source. and the download games won't work.)
